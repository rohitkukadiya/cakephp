Q1. What browsers are supported?

We develop our code to full W3C standards. This means that every mark-up we produce will display perfectly across all major browsers. We test using the latest versions of Google Chrome, Firefox, Internet Explorer 11, Edge, Opera and Safari.

Q2. What format can I send designs in?

We can use files supplied in PSD, AI. INDD, PNG and Sketch formats as a basis for producing a mark-up. If you have a file in a different format, then please contact us so we can find a solution.

Q3. How many revisions do I get?

We offer as many free revisions as needed to make sure you are 100% happy with the code you receive. This means that you are assured of receiving the results you expected.

Q4. Do you charge for extra JavaScript features?

We include the cost of minor JavaScipt development such as image rollovers, sliders and pagination in to the cost of the project, and do not charge extra for these small additions.
